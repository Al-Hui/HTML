# HTML
## food_ai_diary.html
Зайди во вкладку “API / настройки”, вставь свой API key, выбери модель (например gpt-4.1-mini).

Во вкладке “Фото & добавление”:
	•	сделай фото еды (или выбери из галереи),
	•	нажми “Распознать с AI”,
	•	подредактируй при желании название и ккал,
	•	нажми “Добавить в дневник”.
Во вкладке “Журнал” смотри список блюд.
Во вкладке “Аналитика” — графики и суммарные калории.
